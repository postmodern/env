## Description

Provides a Ruby interface to common environment variables, used on both
Windows, Linux, BSD and OSX.

## Features

* Provides access to:
  * `PATH`
  * `LANG`
  * `COLOMNS`
  * `EDITOR`
  * `HOME`
  * `HOSTNAME`
  * `HOSTTYPE`
  * `LINES`
  * `MACHTYPE`
  * `MAIL`
  * `OSTYPE`
  * `SHELL`
  * `SSH_ASKPASS`
  * `TERM`
